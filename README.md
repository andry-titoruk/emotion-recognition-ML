Проєкт Emotion Recognition ML — це система комп’ютерного зору, що виконує розпізнавання емоцій людини в режимі реального часу за допомогою вебкамери та нейромережевої моделі глибокого навчання. Застосунок захоплює відеопотік із камери, виконує детекцію обличчя за допомогою алгоритму Haar Cascade (OpenCV), після чого кожне знайдене обличчя приводиться до необхідного формату та передається на вхід згорткової нейронної мережі для класифікації.

Для класифікації використовується попередньо натренована модель Mini-XCEPTION, навчена на датасеті FER2013. Модель працює з одноканальними зображеннями розміром 64×64 пікселів. Перед подачею в модель область обличчя вирізається з кадру, конвертується у відтінки сірого, масштабується до потрібного розміру, нормалізується (значення пікселів приводяться до діапазону 0–1) та перетворюється у тензор форми (1, 64, 64, 1). Після цього виконується передбачення, і клас із максимальною ймовірністю відображається поверх відеопотоку.

Система визначає сім базових емоцій: Angry, Disgust, Fear, Happy, Neutral, Sad та Surprise. Обробка відбувається локально без використання серверної інфраструктури. Архітектура рішення побудована на Python із використанням TensorFlow/Keras для інференсу моделі, OpenCV для роботи з відео та детекції облич, а також NumPy для обробки даних. Проєкт може бути розширений шляхом додавання збереження статистики емоцій, інтеграції з веб-інтерфейсом, підключення більш сучасних моделей детекції облич або оптимізації продуктивності для вбудованих систем.
